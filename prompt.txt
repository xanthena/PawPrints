Task

the main task is to take an input of a raw_json and give an output as final_timeline.json

raw_json: PawPrints\jsons\raw_json.json

Overall Pipeline
raw_json
    │
    ▼
clean_results()
    │
    ▼
normalize_results()
    │
    ▼
detect_state_changes()
    │
    ▼
merge_consecutive_events()
    │
    ▼
score_events()
    │
    ▼
generate_event_json()

Each stage transforms the data a little.

Stage 1 — Clean the JSON

Some outputs are malformed.

Example:

{
    "raw_output": "```json ... ```"
}

Others already contain

{
    "activity":"sleeping",
    ...
}

Function

clean_results(results)

Responsibilities

for every frame

    if raw_output exists

        extract JSON

        parse it

        replace result

    if parsing fails

        mark frame invalid

return cleaned_results

Output

Every frame now has exactly

activity
confidence
interaction
summary
objects

Function

detect_state_changes(results)

Pseudo

current = None

for frame

    if activity != current

        create event

        current = activity

Notice

We don't know event duration yet.

Only beginnings.

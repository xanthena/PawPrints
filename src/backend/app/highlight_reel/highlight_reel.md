# Highlight reel

This module reads a final vision-model event timeline, selects a few strong and
varied moments, cuts them from the source video, and joins them into one MP4.

## Model and timeline routing

The routing contract mirrors `vision_model/model_router.py`. A caller can pass
an explicit primary/fallback pair, while a future API or UI can use the same
function arguments without changing module internals.

Resolution order:

1. Explicit `--timeline` path, when supplied.
2. Explicit `--primary-model` and `--fallback-model` arguments.
3. `HIGHLIGHT_MODEL_PRIMARY` and `HIGHLIGHT_MODEL_FALLBACK`.
4. Upstream `VISION_MODEL_PRIMARY` and `VISION_MODEL_FALLBACK`.
5. Defaults: Gemini primary and Qwen fallback.

Supported model identifiers match the vision router: `gemini`, `qwen`,
`claude`, and `openai`. Model paths follow
`final_timeline_<model>.json`, so adding another supported timeline does not
require a local/cloud mapping. If the primary file is absent, the fallback is
used once. Set fallback equal to primary (or configure it as blank through the
environment) to disable fallback.

`--timeline` supports model-neutral output names and timelines generated by
other pipelines. The manifest records `timeline_model` and
`timeline_fell_back` for traceability.

## Selection behavior

- Importance is evaluated relative to the scores in the current timeline.
  There is no fixed minimum such as `importance >= 9`.
- Up to five clips are selected by default.
- Each clip is at most ten seconds by default.
- Repeated activities, interactions, objects, overlapping ranges, and nearby
  times are penalized so one repetitive scene does not fill the reel.
- The chosen clips are returned to chronological order before rendering.

## Run

From `src/backend`:

```powershell
python -m pip install -r app/highlight_reel/requirements.txt
python -m app.highlight_reel
```

Useful options:

```powershell
python -m app.highlight_reel --primary-model qwen
python -m app.highlight_reel --primary-model claude --fallback-model qwen
python -m app.highlight_reel --timeline C:\path\to\final_timeline_custom.json
python -m app.highlight_reel --max-clips 4 --max-clip-seconds 8
python -m app.highlight_reel --ffmpeg C:\path\to\ffmpeg.exe
```

The default output directory is `src/results/highlight-reel`, containing:

- `highlight_reel.mp4`
- `highlight_reel_manifest.json`, with selected events, times, scores, and
  reasons for selection.
